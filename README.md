# Student Management System (Python)

A simple Python-based Student Management System that allows users to manage student records through a command-line interface. The application stores data in a text file, making records persistent even after the program is closed.

## Features

* Add new students
* Delete existing students
* Update student information
* Search students by ID or name
* Display all student records
* Validate student grades (0–100)
* Store data permanently using file handling

## Technologies Used

* Python
* File Handling
* Lists
* Dictionaries
* Functions

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/talhareads/student-management-system-python.git
```

2. Navigate to the project folder:

```bash
cd student-management-system-python
```

3. Run the program:

```bash
python main.py
```
